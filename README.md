# Vue Electron Boilerplate
This boilerplate included with Vuetify , Vuex , Router for a basic project structure
## How to Run it

### Get dependencies

First thing you need to get all dependencies first with `yarn`

## Run

- **Web Mode**

    `yarn start`

- **Desktop Mode** 
    
    `yarn electron:serve`

## Build

- **Web Mode**

    `yarn build`

- **Desktop Mode** 
    
    `yarn electron:build`
